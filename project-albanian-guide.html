Albanian Guide
Website

A bilingual tourism guide for the Mati region of Albania featuring an interactive tour booking form with date validation — deployed live via GitHub Pages.

HTML5 CSS3 JavaScript GitHub Pages

Overview

About the Project

This project was created as a college assignment to showcase the Mati region of Albania as a travel destination. The site is completely bilingual, celebrating Albanian culture and inviting visitors to explore the region’s natural attractions, historical sites, and local experiences.

The website features an interactive tour booking system with comprehensive front-end validation, allowing potential tourists to select dates and submit booking inquiries. The design emphasizes visual storytelling with high-quality imagery and engaging content that captures the essence of the region. Upon successful booking, users receive a confirmation page, providing a complete user journey from discovery to booking.

Features

What It Does

  • Multi-page, fully responsive website with seamless navigation
  • Interactive tour booking form with comprehensive JavaScript date validation
  • Client-side form validation ensuring users select valid future dates
  • Confirmation page for successful bookings with booking summary
  • Complete content in Albanian language celebrating local culture
  • Responsive design optimized for mobile, tablet, and desktop viewing
  • Deployed as a live website on GitHub Pages for public access
  • SEO-friendly markup and structure for better search engine visibility
